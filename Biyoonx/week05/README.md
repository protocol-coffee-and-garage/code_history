# week5: BubbleSort, SelectionSort, InsertionSort

## BubbleSort
- 데이터의 인접 요소끼리 비교하고, swap 연산을 수행하며 정렬
- 시간복잡도 : O(n<sup>2</sup>)


## SelectionSort
- 대상에서 가장 크거나 작은 데이터를 찾아 선택을 반복하면서 정렬
- 시간복잡도 : O(nsup>2</sup>)


## InsertionSort
- 대상을 선택해 정렬된 영역에서 선택 데이터의 적절한 위치를 찾아 삽입하면서 정렬
- 시간복잡도 : O(nsup>2</sup>)
